# 常用语法

c 使用的版本 c99

## 基础语法
### 数组

```
int a[10] = { 91, 5, 8, 4, 62, 47, 14, 52, 65, 74};
```

### 结构体

```c
typedef struct { double a,b;} T;

/* 1 */
T t = {.a=10.0,.b=20.1};

/* 2 */
T a;
a.a = 10;
a.b = 20;
```

### 枚举
枚举默认从零开始技术，可以
```
typedef enum {LEFT, RIGHT=3, a, b} lr;

int main() {
	lr t=LEFT, r=RIGHT, t1=a;
	printf("%x %x %x\n", t, r, t1);// 0 3 4
}
```
